        API to create new individual enrollment. Reference: https://developer.salesforce.com/docs/atlas.en-us.loyalty.meta/loyalty/connect_resources_enroll_individual_member.htm

        Parameters:
        firstName - First name of the user
        lastName - Last name of the user
        email - Email ID of the user
        additionalContactAttributes - Map of contact related attributes like phone number
        emailNotification - If user prefers to have email notifications.
        memberStatus - Status of the enrolled member MemberStatus
        createTransactionJournals - Indicates whether to create the transaction journal records (true) or not (false)
        transactionalJournalStatementFrequency - The frequency at which transaction journal statements must be delivered to the member.
        transactionalJournalStatementMethod - The method used to deliver transaction journal statements to the member.
        enrollmentChannel - The channel used to by loyalty program member to enroll in to the loyalty program.
        canReceivePromotions - Indicates whether the loyalty program member can receive promotions.
        canReceivePartnerPromotions - Indicates whether the member can receive partner promotions for the program (true) or not (false).
